# ZENITH | Luxury Editorial

A minimalist fashion editorial website showcasing the ZENITH collection, featuring structural minimalism and sustainable textiles.

## Description

ZENITH is a luxury fashion brand focused on engineering longevity through innovative design and materials. This website presents the brand's philosophy, collection, and laboratory processes in a clean, geometric layout.

## Features

- **Hero Section**: Large-scale typography with brand coordinates
- **Collection Gallery**: Grid-based image showcase with hover effects
- **The Lab**: Information about sustainable production and testing
- **Navigation**: Fixed header with smooth scrolling
- **Footer**: Links to social media and brand information
- **Responsive Design**: Adapts to different screen sizes

## Technologies Used

- HTML5
- CSS3 (with custom properties and transitions)
- Webkit-specific text effects for visual styling

## How to Run

1. Clone or download the repository
2. Open `index.html` in a modern web browser
3. For best experience, serve the files using a local HTTP server to avoid CORS issues with local file access

### Using Python (recommended for local development)

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

### Using Node.js (if available)

```bash
npx serve .
```

## Browser Support

- Chrome/Chromium (full support for text-stroke effects)
- Safari (full support)
- Firefox (fallback text color, may not show hollow text effects)
